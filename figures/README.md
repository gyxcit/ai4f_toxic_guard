# Figures

This folder contains the four publication figures used in the paper, each
provided in two formats:

- **`.png`** — raster format at 300 DPI, suitable for slides or web

All figures are generated by `notebooks/03_analysis_and_figures.ipynb`.

---

## Files

| File | Paper reference | Description |
|---|---|---|
| `fig1_cvar.pdf` / `.png` | Figure 1 (Section 6.3) | CVaR$_{10\%}$ and CVaR$_{25\%}$ of excess capital over the passive baseline, per configuration |
| `fig2_winrate.pdf` / `.png` | Figure 2 (Section 6.3) | Episode-level win rate with 95% Clopper-Pearson confidence intervals |
| `fig3_regime.pdf` / `.png` | Figure 3 (Section 7) | Heatmap of mean excess capital by dominant market regime (calm, normal, stressed) |
| `fig4_corr.pdf` / `.png` | Figure 4 (Section 7) | Pairwise Pearson correlations between the four toxicity scores |

---

## Key visual takeaways

- **Figure 1** — R5_volsize is the only configuration with a strictly positive
  CVaR$_{10\%}$ (+$4.00). All other configurations are negative; R7_all has
  the worst tail loss (-$63.09).

- **Figure 2** — R5_volsize achieves a perfect 15/15 win rate across all
  windows; R6_lvr+spread and R7_all underperform the toxicity-blind R1
  baseline.

- **Figure 3** — R5_volsize and R1_baseline perform comparably in calm and
  normal regimes; the gap (+$36.65) emerges entirely in stressed regimes.

- **Figure 4** — `tox_volsize` is nearly collinear with `tox_spread`
  (r = 0.985) by construction; this redundancy explains the instability of
  R7_all.

## Reproducibility

To regenerate all figures from the per-run results:

```python
jupyter notebook notebooks/03_analysis_and_figures.ipynb
```

The notebook reads `results_final.csv`, `cvar_results.csv` and
`actions_log_R5.csv` from the `results/` folder (released upon acceptance)
and writes the four figures back into this folder.

For convenience, the published versions of the figures are committed to the
repository so readers can inspect them without re-running the analysis.

---

## Styling

All figures use:

- A serif typeface to match the LaTeX paper body
- 300 DPI rasterisation
- Consistent colour palette: green for R5 (highlight), red for negative
  outcomes, blue for neutral PPO baselines
- No promotional annotations or arrows; emphasis is conveyed by colour and
  bordered cells (e.g., R5 highlight in Figure 3) only
